What's a Preset?
With presets, you can save the currently selected cases, and afterwards select all of them at once in one click.
- Click Preset Name to select its cases
- Doubleclick preset name to rename
- Click set to save currently selected cases in the preset
- Click Reset to reset the preset's stored data and name

What's the inverses ON/OFF button?
Inverses ON will include the inverse of your selected cases, so selecting AG also selects GA.

What is cases before break?
The amount of cases you will be shown, before having a break. 0 means no breaks.

What is cases target time?
After practicing, you can repeat the cases slower than your target time. 3s by default.
